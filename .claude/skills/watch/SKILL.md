# /watch — Position Monitoring

Dispatch the Monitor agent to check all open positions and recommend actions.

## Instructions

Launch the Monitor agent (`monitor.md`) to:
1. Review all open positions
2. Check for material news or price movements
3. Evaluate risk status
4. Recommend HOLD, CLOSE, or SIZE UP for each position
5. Flag any urgent alerts
